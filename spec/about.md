## 🌱 这个博客有什么？

- 📝 **学习笔记**：记录前端、后端、工具使用等学习心得
- 🛠 **项目分享**：展示自己开发或参与的小项目

- 🌸 **生活碎片**：偶尔也会记录一些日常的小确幸

## 🛠 技术栈

本站基于 [Astro](https://astro.build/) 框架构建，主题为 [Mizuki](https://github.com/matsuzaka-yuki/Mizuki)：

- **框架**：Astro
- **样式**：Tailwind CSS
- **评论**：Twikoo
- **搜索**：Pagefind
- **内容**：本站采用**内容分离**模式
- **代码高亮**：Expressive Code
- **数学公式**：KaTeX
